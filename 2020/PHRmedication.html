PHR 居家用藥資訊標準化互通

包含的子系統: PHR FHIR server、民眾端系統、醫護系統、數據分析端系統
PHR FHIR server 由主辦單位提供。
聯測參與單位扮演民眾端系統、醫護系統、或數據分析系統腳色(client 端系統)。


藥物處方上傳到 PHR FHIR server

  1. 民眾端系統或醫護系統上傳藥物處方到 FHIR server

注:民眾提供授權及 PHR Patient.id 給醫護系統

注:藥物處方可不提供個資、處方開立單位人員、藥事服務人員資訊(以確保病人隱私)。後續將以電子簽章技術確認釋出處方正確性及完整性


民眾上傳用藥紀錄到 FHIR server

  1. 民眾上傳用藥紀錄到 FHIR server

藥物處方及用藥紀錄查詢

  1. 醫護系統或數據分析系統調閱民眾藥物處方及用藥紀錄

注:民眾提供授權及 PHR Patient.id 給醫護系統及數據分析系統


PHR 藥物處方及用藥紀錄規格聯結。
