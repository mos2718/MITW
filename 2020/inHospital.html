機構內系統資訊標準化互通

包含的子系統: FHIR server、機構個案管理系統、醫療照護者 APP 或網頁系統


與院內 FHIR server 互通的資訊

機構個案管理系統

  1. 機構所屬病人增刪改查
  2. 藥物處方及生理量測結果增刪改查

醫療照護者 APP 或網頁系統

  1. 查詢機構所屬病人 (機構所屬病人可能很多，可設定查詢參數)
  2. 查詢特定病人之藥物處方及生理量測結果(使用 patient id 及生理檢測類別做查詢)
  3. 上傳及查詢特定病人之用藥紀錄(增刪改查)
  4. 上傳及查詢特定病人之生理監測結果(增刪改查)，查詢結果需在 client 端洽當呈現(如列表或曲線圖呈現)
